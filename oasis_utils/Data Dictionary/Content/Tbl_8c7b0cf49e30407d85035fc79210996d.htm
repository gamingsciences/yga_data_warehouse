Owner  dbo 
Definition  Persistent temporary table used for interim processing of slot-machine data for Wisconsin Net Win extracts. All data from WIS_NETWIN_xxx_EXTRACT tables relative to this table as a slot-master table for the relevent date ranges, are populated and massaged and inserted/updated in WIS_NETWIN_EXTRACT_TMP, which is the final source of the data extract
