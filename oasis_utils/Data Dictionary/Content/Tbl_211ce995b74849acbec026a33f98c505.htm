dbo.DM_MeterTable 
Logical Entity Name  DM_MeterTable 
Default Table Name  DM_MeterTable 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  In Diagnostic Monitor, when the User clicks on a machine in the "Statistics" tab, Diagnostic Monitor inserts a copy of all the current meters and some machine information into the DM_MeterTable table. This is so you can go back and see what the meter values were for a specific machine you are looking at.
