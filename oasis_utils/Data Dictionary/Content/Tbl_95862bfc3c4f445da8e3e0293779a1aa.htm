dbo.BB_SlotType 
Logical Entity Name  BB_SLOTTYPE 
Default Table Name  BB_SlotType 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Main table to keep track of the configurations of the slot types. For each unique slot Type ID, the machine can have several configurations during its life. The revision number from the machine identifies each configuration.
