Definition  Stores all maintenance transactions sent up to the system from the Sentinel. This includes Failure codes, Repair codes and System-generated failure codes.

Threshold    int  YES  An overall threshold indicating how many times a repair or failure code will have to be entered into the system to generate a work order. For example, if the threshold is set to 10 then it will take 10 of the same failure codes to create a workorder. 
ThresholdTW    int  YES  A time-specific threshold indicating how many times a repair or failure code will have to be entered into the system in a certain time-frame to generate a work order. For example, if the threshold is set to 10 within 8 hours then it will take 10 of the same failure codes to be done in an 8 hour time period to create a work order. 
TimeWindow    int  YES  A window of time designated for work orders to be created based on the ThresholdTW. For example, if the threshold is set to 10 and the TimeWindow is set to 8 hours then it will take 10 of the same failure codes to be done in an 8 hour time period to create a work order.
