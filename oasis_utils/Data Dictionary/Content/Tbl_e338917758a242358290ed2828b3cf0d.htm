dbo.CDS_Expenses Columns 
ColumnName  Domain  Datatype  NULL  Definition 
IDType    char(1)  NO  This indentifies the account type. 'A' = Account, 'P' = Player, 'G' = Group. 
Meta_ID    int  NO  The Meta ID could be either a Player ID or a Group ID. There could possibility be a Group ID and Player ID with the same value. The ID Type identifies the difference between the two. 
Description    varchar(20)  NO  The description of the expense. 
ExpenseType_ID    tinyint  NO  The unique system-generated sequential number use to identify the Expense. This ID is created in OASIS Administrator. 
Amount    numeric(17, 8)  NO  Unit amount of the expense for non-percentage type expenses. 
Percentage    numeric(11, 8)  NO  For certain types of criteria, the percentage value of the expense. 
ExpenseCriteria_ID    tinyint  NO  For certain t ypes of expenses (Per Player Played), this is the criteria that refines the expense type (such as "Total In" or "Total Out"). 
HardCost    char(1)  NO  Flag to indicate that the expense is a hard cost. 
ActualTotal    money  NO  The actual amount of the expense for the group.
