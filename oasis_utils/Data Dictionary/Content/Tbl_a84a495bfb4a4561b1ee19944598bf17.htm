dbo.BadDeltaMeterOriginal 
Logical Entity Name  BadDeltaMeterOriginal 
Default Table Name  BadDeltaMeterOriginal 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Stores the meters that were determined to be bad during the original revenue calculation process for a given Auditdate. The meters that where determined to be bad during the latest revenue calculation process are kept in BadDeltameter.
