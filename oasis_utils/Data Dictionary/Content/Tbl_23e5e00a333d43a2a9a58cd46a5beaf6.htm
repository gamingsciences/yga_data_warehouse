dbo.SpeedMediaEventCustomMessageText Columns 
ColumnName  Domain  Datatype  NULL  Definition 
CustomMessageTextID    int  NO  PK - Int value that is automatically generated (identity). Each unique number represents one custom text message. 
EventID    int  NO  Int value that is foreign keyed to an identity field in the SpeedMediaEvent table. Each number identifies an event. 
ScreenTypeID    tinyint  NO  TinyInt value that is generated by ATI and defined in the ScreenType table. Values can include, among others, Attractor, In Play-Carded, Welcome, Congratulations and Birthday. 
x1    smallint  NO  SmallInt value which represents the coordinates that define where on the horizontal axis the text messages are displayed on the screen. 
x2    smallint  NO  SmallInt value which represents the coordinates that define where on the horizontal axis the text messages are displayed on the screen. 
y1    smallint  NO  SmallInt value which represents the coordinates that define where on the vertical axis the text messages are displayed on the screen. 
y2    smallint  NO  SmallInt value which represents the coordinates that define where on the vertical axis the text messages are displayed on the screen. 
MessageText    varchar(255)  NO  VarChar(255) value which is the actual text to be displayed. 
FontColor    varchar(6)  NO  VarChar(6) value which is a Hexidecimal representation of the font color. 
FontSize    smallint  NO  SmallInt value indicating the size, in points, of the font. 
FontBold    bit  NO  A bit value indicating if the text is in bold type or not. 
FontUnderline    bit  NO  A bit value indicating if the text is underlined or not. 
FontItalics    bit  NO  A bit value indicating if the text is italicized or not. 
FontName    varchar(25)  NO  VarChar(25) value representing the name of the font selected. 
JustificationTypeID    tinyint  NO  TinyInt value which represents the left, right or center text justification as outlined in the SpeedMediaJustificationType table. 
DropShadow    bit  NO  A bit value indicating if there is a drop shadow or not. 
DisplayOrder    tinyint  NO  When more than one custom message tag is user-defined for a screen, this indicates the display order of those custom messages 
AllMediaItems    bit  NO  Yes/No Bit Indicates the Custom Message Tag is displayed for all Media items in this Media Package 
PackageOrientationTypeID    tinyint  NO
