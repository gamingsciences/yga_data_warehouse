SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.

SenOption    int  NO  Bit mask containing sentinel options for a specific EGM 
SenOption2    int  NO  Bit mask containing additional sentinel options for a specific EGM 
Port    varchar(10)  YES  Port associated to a specific EGM 
AddCurCC    bit  NO  Tells the mid tier weather or not to add the current Cancelled Credit meter when sending meters to the database 
AddCurJP    bit  NO  Tells the mid tier weather or not to add the current Jackpot meter when sending meters to the database 
AddCurHandPay    bit  NO  Tells the mid tier weather or not to add the current hand pay meter when sending meters to the database 
SentinelVersion    nvarchar(128)  YES  Version of the software of a specific sentinel 
SentinelChecksum    bigint  YES  Checksum for the software version of a specific sentinel 
Lockout    bit  NO  Identifies whether a particular sentinel is locked or not. 0=No, 1=Yes. 
LockoutStateChangedDate    datetime  YES  Identifies when the lockout state was last changed. 
SentinelIP    varchar(39)  YES  For sentinel III, the IP address of the sentinel. 
SentinelOS    varchar(10)  YES  For sentinel III, the OS version that the sentinel is running. 
OCR    char(10)  YES  10 digit number that uniquely identifies a player or employee card in the CDS_Card table that is currently inserted in a machine.  
SentinelFeatureFlag    bigint  NO  Bit mask containing additional Feature Flags for a specific EGM. The values are defined the SentinelFeatureFlag table.
