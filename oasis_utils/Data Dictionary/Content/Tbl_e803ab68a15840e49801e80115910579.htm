dbo.SpeedMediaEventTrigger 
Logical Entity Name  SpeedMediaEventTrigger 
Default Table Name  SpeedMediaEventTrigger 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  This table links the SpeedMediaEventType table to the SpeedMediaTriggerType table and identifies which trigger belong with which event. For example, the Coin IN and Coin Loss belong to the Splashdown Countdown and Ricochet Rewards event types. 
Note   
PreSQL   
DDL Code  CREATE TABLE dbo.SpeedMediaEventTrigger(
     EventTypeID     tinyint     NOT NULL,
     ID              tinyint     NOT NULL,
     CONSTRAINT pkc_SpeedMediaEventTrigger_EventTypeID PRIMARY KEY CLUSTERED (EventTypeID, ID),
     CONSTRAINT fk_SpeedMediaEventTrigger_SpeedMediaTriggerType_TriggerTypeID FOREIGN KEY (ID)
     REFERENCES dbo.SpeedMediaTriggerType(ID),
     CONSTRAINT fk_SpeedMediaEventTrigger_SpeedMediaEventType_EventTypeID FOREIGN KEY (EventTypeID)
     REFERENCES dbo.SpeedMediaEventType(EventTypeID)
)
