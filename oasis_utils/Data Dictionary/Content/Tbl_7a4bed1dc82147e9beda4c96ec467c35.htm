dbo.SpeedMediaEvent Columns 
ColumnName  Domain  Datatype  NULL  Definition 
EventID    int  NO  Uniquely identifies an Event in the SpeedMediaEvent table. 
EventName    varchar(40)  NO  VarChar(40) text value that describes the event. 
EventTypeID    tinyint  NO  TinyInt value that is generated by ATI and defined in the SpeedMediaEventType table. Values can include, among others, Splashdown Countdown, Ricochet Rewards, Jackpot Announcement, CDM Promotion and Totalizer Announcement. 
AllMachine    bit  NO  Bit value that indicates if all slot machines are included in the event or not. 
AllPlayer    bit  NO  Bit value that indicates if all players are included in the event or not. 
Ready    bit  NO  Bit value that indicates if the event is active or not. 
Status    tinyint  NO  TinyInt value that is generated by ATI and defined in the SpeedMediaEventStatus table. Values can include, among others, Not Scheduled, Scheduled, In Progress, Complete and Error. 
VendorID    smallint  NO  SmallInt value that is generated by ATI and defined in the SMAVendor table. Each number identifies a vendor. 
VendorLocationID    int  YES  Int value that is generated by ATI and defined in the SMAVendorLocation table. Each number identifies a vendor location. 
LicenseID    int  NO  Int value that is generated by ATI and defined in the SMAPromotionLicense table. Each number identifies a license. 
AwardFrequencyType    tinyint  NO   
AwardsPerFrequency    tinyint  NO   
GlobalEventID    varchar(32)  NO  Global Event ID for OmniCard properties and concurrency 
IsGlobal    bit  NO  Yes/No Bit Indicates Event Is for all OmniCard Casino Properties
