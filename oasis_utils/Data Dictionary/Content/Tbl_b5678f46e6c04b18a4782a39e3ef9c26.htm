dbo.JackpotAnnouncerEvent Columns 
ColumnName  Domain  Datatype  NULL  Definition 
EventID    int  NO  PK - Int value that is foreign keyed to an identity field in the SpeedMediaEvent table. Each number uniquely identifies a event for Jackpot Announcement. 
TriggerTypeID    tinyint  NO  TinyInt value generated by ATI and defined in the SpeedMediaTriggerType table. It describes each Trigger as being of a specific type (e.g. Coin In, Coin Loss, All Jackpots, Attendant Paid Jackpot, etc.). 
TriggerThresholdMin    money  NO  Money value indicating the minimum trigger threshold for which jackpots should be displayed. 
TriggerThresholdMax    money  NO  Money value indicating the maximum trigger threshold for which jackpots should be displayed. 
IncludeCardedPlayers    bit  NO  Bit value indicating whether or not the announcement will display carded players on the Sentinel III. 
IncludeNonCardedPlayers    bit  NO  Bit value indicating whether or not the announcement will display noncarded players on the Sentinel III.
