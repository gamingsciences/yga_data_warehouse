dbo.JF_FloorDebug 
Logical Entity Name  JF_FLOORDEBUG 
Default Table Name  JF_FloorDebug 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  The Poller inserts into this table (via the stored procedure RT_JFPEND1), a record of every item concerning a particular slot machine transaction, from the time the jackpot occurs, until the transaction is fully processed, saved, and removed from CDS_JFPEND. Basically an audit trail that can be filtered by date ranges and record numbers.
