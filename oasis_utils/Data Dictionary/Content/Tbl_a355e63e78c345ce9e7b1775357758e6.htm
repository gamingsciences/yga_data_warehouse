dbo.CP_MARKETING_UN 
Logical Entity Name  CP_MARKETING_UN 
Default Table Name  CP_MARKETING_UN 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  This table is for unnumbered Coupons. It is slightly different than CP_MARKETNIG, because in an unnumbered Coupon series, every coupon has the SAME validation number. Every time someone redeems a coupon it writes a record to the Detail table and increments the field (NumRedeemed) in this table.
