dbo.JF_AuthTicket 
Logical Entity Name  JF_AUTHTICKET 
Default Table Name  JF_AuthTicket 
Physical Only  NO

Owner  dbo 
Definition  Lookup table for types of Jackpot/Fill amount authorizations. Stores information about each type of authorization such as a text description and how many people are required for authorization.
