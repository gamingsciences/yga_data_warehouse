dbo.BB_RevenueDTD 
Logical Entity Name  BB_REVENUEDTD 
Default Table Name  BB_RevenueDTD 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Main table to keep track of the electronic Drops-To-Date (DTD) Dollars Amounts of the Slot Machines on the OASIS system such as: Coins, Bills, Tickets, Coupons, and so on for period DTD.

SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.
