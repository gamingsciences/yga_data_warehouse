ExpiredDateTime    datetime  YES  The date and time the coupon expired.

VoidedID    int  YES  OASIS User ID of the user who voided the coupon. 
DetailCount    int  NO  Not used. 
CouponsubType    varchar(5)  YES  Coupon SubType: Prize, Comp, Cash. 
PrizeOrCompID    int  YES  This is the entry into CDS_PRIZE table, if it is a Prize or Comp or cash. This is the ID that ties back to CDS_Prize.

Player_id    int  YES  Unique ID number given to each player added via the Player Notebook in Super-PlayMate or PitBOSS.
