dbo.CDS_CardInfo Columns 
ColumnName  Domain  Datatype  NULL  Definition 
PlayerID    int  NO  If the pin information is for a player uniquely identifies a player in the CDS_Player table. If the pin information is for a user the negative of the UserID is stored. 
LockType    tinyint  NO  Uniquely identifies type of lock in the PlayerAccoutLockType table. 
UserID    int  YES  If the pin information is for a User and not a player uniquely identifies the User_ID in the CDS_User table otherwise value will be null. 
PlayerPIN    binary(32)  YES  The encrypted pin number. 
PINUpdatedDatetime    datetime  NO  The date and time the pin was last updated. 
LockedByApplicationID    smallint  YES  If the account is locked uniquely identifies the application that locked it in the CDS_Application table. 
LockedBySlotMastID    int  YES  If the account is locked uniquely identifies the slot machine that locked it in the SlotMast table. 
LastLockedDatetime    datetime  YES  The date and time the account was last locked. 
BadPINCount    tinyint  NO  The number of bad pins entered into the system since the last good pin was entered. 
Seed    bigint  YES  Random value used for encryption of the PlayerPin.
