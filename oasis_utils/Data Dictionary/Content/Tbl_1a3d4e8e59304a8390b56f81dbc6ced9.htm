Definition  This is a very important table, possibly the most important table in the Coupons application because it is the link between the specifications in the following tables: CP_Offers, CP_Coupons, and the Tables sets of the Individual Coupons (CDS_TICKET, CP_MARKETING, CP_MARKETING_UN [and their detail tables]). This table is also important for imported Coupons. If data is imported, this table holds the amout and the prize ID for an Individual Coupon. Unlike regular Coupons, imported coupons can have varying amounts and different prizes. When the User reserves validation numbers, these numbers are written into the DATABASE for the first time, in this table. When entries are created for individual coupons, the data is written back to the field RECORD_IX.
