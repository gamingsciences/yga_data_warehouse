EventStartAccumulationTypeID    tinyint  NO  TinyInt value which defines how the system will collect and accumulate statistics to meet the promotion trigger threshold. 
DailyBudget    int  YES  Int value which translates into a calendar day limit for the corresponding RR event. It is not a required field. "Daily" refers to a calendar day from 12:00:00 am to 11:59:59 pm. When a user defines the daily budget for a specific Ricochet Rewards promotion, both start and celebration awards count towards the daily budget amount for each occurrence. The daily budget value should never be exceeded. Once the daily budget has been reached during the day for a particular RR event, the start trigger will no longer occur and, if a player meets the start trigger threshold, no awards will be issued. The daily budget takes precedence over the number of ricochets that can occur per event occurrence. Unused monies from the daily budget will not roll over to the next day.
