dbo.PB_RevenueDTD Columns 
ColumnName  Domain  Datatype  NULL  Definition 
Timestamp    datetime  NO  The date and time when the record was inserted or last updated. 
Casino_ID    tinyint  NO  Unique casino ID used in conjunction with mulit-casino or multi-site casinos. Default: dbo.DEF_CASINOID. 
Location_ID    int  NO  The location ID where the transaction was performed. 
AuditDate    datetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
Shift_ID    char(1)  NO  Current Cage shift. 'D' =Day, 'G' = Grave, 'S' = Swing. 
OpenerChips    numeric(15, 2)  NO  Sum of all Opener types for a given location, date, and shift. 
OpenerIOU    numeric(15, 2)  NO  Sum of all opener types with an inventory item of MISC for a given location, date and shift. 
CloseChips    numeric(15, 2)  NO  Sum of all closer types for a given location, date and shift in pb_document. 
CloseIOU    numeric(15, 2)  NO  Sum of all Closer types with an inventory item of MISC for a given Location, date, and shift. 
Fills    numeric(15, 2)  NO  Sum of all the fills for the given location, date, and shift. 
Credits    numeric(15, 2)  NO  Sum of all the credits for the given location, date, and shift. 
MarkerIssues    numeric(15, 2)  NO  Sum of all the new markers issued for the given location, date, and shift. 
MarkerPaymentsCash    numeric(15, 2)  NO  Sum of all payments types where the inventory item name is cash. 
MarkerPaymentsChips    numeric(15, 2)  NO  Sum of all payments types where the inventory item name is chips. 
MarkerTransfers    numeric(15, 2)  NO  Sum of all the transfers for the given location, date, and shift. 
CashDrop    numeric(15, 2)  NO  Total value of all the Cash and Chips for a location for a given date and shift. 
DropTotal    numeric(15, 2)  NO  Original Amount on the document type drop created for that gaming date, shift, and location. 
StatisticalDrop    numeric(15, 2)  NO  (DropTotal + MarkerIssues) - MarkerPaymentsCash. 
StatisticalWin    numeric(15, 2)  NO  CloseChips - OpenerChips - Fills + Credits + StatisticalDrop - MarkerPaymentsChips. 
RowVersion    timestamp  NO
