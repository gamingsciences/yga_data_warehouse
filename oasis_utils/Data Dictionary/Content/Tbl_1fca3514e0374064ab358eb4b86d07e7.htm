Logical Entity Name  WIS_NETWIN_CUR_EXTRACT 
Default Table Name  WIS_NETWIN_CUR_EXTRACT 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Current revenue period net-win data from bb_Revenue (Period_Id = 4). This table is deleted and repopulated with each run of the stored proc WIS_NETWIN_EXTRACT. After repopulation, this data is combined with data ifromm other WIS_... tables for regulator reports. It is used for Wisconsin's regulators and is exclusively for Wisconsin casinos.
