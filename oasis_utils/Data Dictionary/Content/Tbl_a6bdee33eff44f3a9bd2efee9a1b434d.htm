dbo.BB_BadPBT Columns 
ColumnName  Domain  Datatype  NULL  Definition 
AuditDate    smalldatetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.  
SlotNumber    int  YES  Slot Number refers to the number of the gaming machine. This number corresponds to the number on the machine's cabinet. 
Trans_ID    int  NO  Transaction ID from the EFT_TRANS table. 
CasinoCode    char(2)  YES  A component of the Location assigned to a machine. Casino Code can represent a single casino or distinguish between casinos in a multiple casino environment. 
Revision    smallint  YES  Each time a slot machine record is opened, modified in any way, and then saved through BlackBart, the revision number is incremented by +1. 
Timestamp    datetime  NO  Timestamp from the EFT_TRANS table. 
TransDate    datetime  NO  Transaction Date from the EFT_TRANS table. 
EFTType    tinyint  NO  Uniquely identifies the type of EFT transaction in the EFTType table. 
Amount    numeric(15, 2)  NO  Bad Personal Banker transaction amount. 
Denomination    numeric(12, 2)  YES  The monetary value a slot machine is setup to use.
