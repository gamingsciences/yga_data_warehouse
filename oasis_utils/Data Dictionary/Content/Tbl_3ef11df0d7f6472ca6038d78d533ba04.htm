Default Table Name  EFTTransactionPromoDetail 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Detail record of an EFTTransaction which identifies how many points were redeemd from a Promo Event. Provides a breakout of which Issued PBT Promo in the PBTPromoPlayer table is used in which transactions. It’s a detail because a given EFT Transaction could use Promo from multiple PBT Group Issues and/or awards.
