Casino_ID    tinyint  NO  Unique casino ID used in conjunction with mulit-casino or multi-site casinos. Default: dbo.DEF_CASINOID. 
JFPend_ID    int  NO  Unique ID assigned to each transaction as it is inserted into this table. This is an Identity field and the Primary Key. 
TransType    char(1)  NO  Identifies the transaction type which may be either 'J' for jackpot or 'F' for fill. **Note:** 'F' is also used to indicate transaction types for PMFills. 
IsProgressive    char(1)  NO  If the current record is a progressive jackpot then 'Y', otherwise if not progressive then 'N'. 
FloorCard_ID    varchar(10)  NO  The unique 10-digit User card ID number. 
PlayerCard_ID    varchar(10)  NO  The unique 10-digit player card ID number. 
Player_ID    int  NO  Unique ID number given to each player added via the Player Notebook in Super-PlayMate or PitBOSS. 
DpuID    tinyint  NO  Unique ID number for the DPU associated with up to 31Sentinels. 
SenID    tinyint  NO  Unique ID number used to identify the Sentinel located within a slot machine. 
SlotID    int  NO  Casino-specific number assigned to a slot machine. Generally, used to seperate machines based on denomination. For example,, numbers in the 1000's represent nickel machines, numbers in the 2000's represent quarter machines, etc. 
SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.  
Revision    smallint  NO  Each time a slot machine record is opened, modified in any way, and then saved through BlackBart, the revision number is incremented by +1. 
Denomination    numeric(12, 2)  NO  The monetary value a slot machine is setup to use. 
LocationString    char(20)  NO  Used to identify exactly where a slot machine is located on a casino floor. Consists of 4 parts: 1) Casino Code, 2) Area, 3) Section & 4) Location. 
NormalFillAmt    numeric(12, 2)  NO  The default fill amount as defined in BlackBart. 
MaxJackpotAmt    numeric(12, 2)  NO  The maximum jackpot amount that a slot machine will pay. 
RecordAddedDateTime    datetime  NO  Timestamp when the record was added. 
MachineDateTime    datetime  NO  Timestamp representing the time a transaction was sent from the Sentinel. 
FloorCardDateTime    datetime  NO  Timestamp representing the time the User inserted their floorcard into the slot machine and the trasaction was sent from the Sentinel. 
KeypadDateTime    datetime  NO  Timestamp representing the time sends the keypad signal for the slot machine keypad. 
MachineAmt    numeric(12, 2)  NO  Jackpot value sent from slot machine. 
KeypadAmt    numeric(12, 2)  NO  Jackpot value input by the floor person on the slot machine keypad. 
CoinsPlayed    smallint  NO  The actual number of coins played for a specific jackpot. 
CoinInMeter    bigint  NO  Lifetime value for the number of coins inserted into a slot machine. 
CoinOutMeter    bigint  NO  Lifetime value of the number of coins paid out of a slotmachine. 
CoinDropMeter    bigint  NO  The meter value indicating the number of coins dropped into the slot machine at the time of the jackpot. 
JackpotMeter    bigint  NO  Lifetime total value of every jackpot for a particular slot machine. 
Collected    char(1)  NO  Indicates if the jackpot has been collected. 'Y' = Yes, 'N' = No. 
InUseIPAddress    int  NO  Unused field.
