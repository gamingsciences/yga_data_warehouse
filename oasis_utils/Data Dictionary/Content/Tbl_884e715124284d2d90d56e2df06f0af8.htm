OldArea    char(5)  YES  Casinos divide the gaming floor in to different areas. Each area has a unique identifier. OldArea contains the identifier for the old area. 
OldSection    char(5)  YES  Casinos divide the gaming floor in to different areas. Each area is then divided up into small units called sections. Each section is given a unique identifer. OldSection contains the identifier for the old section. 
OldLoc    char(5)  YES  Casinos divide the gaming floor in to different areas. Each area is then divided up into small units called sections. Each section in turn is further divided into smaller units called locations. Each location is then given a unique identifier. OldLoc contains the identifier for the old location.
