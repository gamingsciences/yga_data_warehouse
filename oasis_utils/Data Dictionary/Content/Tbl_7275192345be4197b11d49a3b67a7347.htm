dbo.CDS_APPLICATION Columns 
ColumnName  Domain  Datatype  NULL  Definition 
Application_ID    smallint  NO  Numeric value used to uniquely identify each application in the OASIS system. This is the Primary Key. As new applications are added, the numeric value is incremented by 1. 
AppName    varchar(32)  NO  Official name of application. 
Installed    char(1)  NO  Indiactes whether this application has been installed at a particular property. 'Y' = Yes, 'N' = No. 
DetailLocation    char(2)  NO  Location where the setup and application information may be found. (OA = Oasis Administrator.) Some applications may not have any OASIS Administrator ties. 
DatabaseName    varchar(30)  NO  Name of the database where this Oasis application has been installed. Default database is 'WinOasis' however, individual properties may change database name, or have more than one database (e.g. Training). 
FirstInstalledDate    smalldatetime  YES  First date this application was installed. 
FirstInstalledVersion    varchar(16)  YES  Version of the first OASIS installation. 
CurrentInstalledDate    smalldatetime  YES  Date the last (current) OASIS installation was performed. 
CurrentInstalledVersion    varchar(16)  YES  The current version of OASIS software installed.
