Definition  Summed revenue for each respective revenue period (bb_Revenue.Period_Id) from all the other WIS_... tables over a given date range. This table is deleted and repopulated from data in WIS_NETWIN_STATIC_EXTRACT, WIS_NETWIN_STATIC_EXTRACT, and other WIS_... tables, on each run of the stored proc WIS_NETWIN_EXTRACT. It is the main table that (1) is made to garner all other WIS_ data and (2) directly provide the reports for which this process exists. It is used for Wisconsin's regulators reports and is exclusively for Wisconsin casinos.
