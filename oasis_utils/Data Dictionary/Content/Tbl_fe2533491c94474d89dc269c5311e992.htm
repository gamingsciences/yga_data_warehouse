Definition  The rows in this table facilitate the correlation of Campaigns and Machine Groups. 
Note  This table is the associate entity between Campaign and Machine
Group.

     CONSTRAINT pkc_SMACampaignMachineGroup_CampaignIDMachineGroupID PRIMARY KEY CLUSTERED (CampaignID, MachineGroupID),
     CONSTRAINT fk_SMACampaignMachineGroup_SpeedMediaCampaign_CampaignID FOREIGN KEY (CampaignID)
     REFERENCES dbo.SpeedMediaCampaign(CampaignID),
     CONSTRAINT fk_SMACampaignMachineGroup_SpeedMediaMachineGroup_MachineGroupID FOREIGN KEY (MachineGroupID)
