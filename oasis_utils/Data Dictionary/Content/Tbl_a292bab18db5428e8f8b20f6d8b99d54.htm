dbo.SchedulerInfo Columns 
ColumnName  Domain  Datatype  NULL  Definition 
SchedulerInfoID    smallint  NO  Primary key 
ServiceID    smallint  NO  References ServiceID of the SchedulerServices table. Specifies which service type the schedule refers to 
FrequencyID    tinyint  NO  References FrequencyId from ScheduleFrequency. Specifies whether the job is to be run daily, weekly or monthly. 1=Daily, 2=Weekly, 3=Monthly 
IsActive    bit  NO  Specifies whether the schedule is active. 0 = Inactive, 1 = active 
LastRunDateTime    datetime  NO  DateTime when the job was last run 
IntervalMonths    tinyint  YES  Will be non-zero only if FrequencyID is 3, i.e. monthly. Specifies the interval in months after which the service will run recurringly 
IntervalDays    tinyint  YES  Specifies interval in days at which the job will run if frequency is daily. Specifies interval in weeks at which the job will run if setup to run weekly. Specifies day of month on which job will run when the job is set to run monthly but does not have scheduled days of the week 
ScheduleTime    time(7)  YES  Time at which the job should run. Will be populated in case the job is non-recurring 
IntervalInMins    int  YES  Specifies the interval in minutes at which the job should run, on the days it is scheduled to run 
StartDate    date  NO  Date on which the schedule will start 
EndDate    date  YES  Date on which the schedule will end 
StartTime    time(7)  YES  Time at which the schedule will start, on the days it is scheduled to run 
EndTime    time(7)  YES  Time at which the schedule will end, on the days it is scheduled to run 
ScheduledDescription    varchar(255)  NO  Desciption of the schedule, describing its run frequency 
IsServiceRunning    bit  NO
