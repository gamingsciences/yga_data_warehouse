Default Table Name  BB_Revenue_MG 
Physical Only  NO

Owner  dbo 
Definition  Stores multigame information on the electronic and manual revenue of the Slot Machines on the OASIS system, such as: Coins, Bills, Tickets, Coupons, PersonalBanker, and so on, for all periods (CUR, PTD, MTD, QTD, YTD, LTD).

dbo.BB_Revenue_MG Columns 
ColumnName  Domain  Datatype  NULL  Definition 
SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.  
GameNumber    int  NO  Game number of a multi-game machine. 
AuditDate    smalldatetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
Period_ID    tinyint  NO  A unique number to identify each one of the accounting periods: 3 = DTD (Drop-to-date), 4 =CUR (Current), 5 = PTD (Period-to-date), 6 = MTD (Month-to-date), 7 = QTD (Quarter-to-date), 8 = YTD (Year-to-date), and 9 = LTD (Lifetime-to-date). 
Game_ID    int  NO  Corresponds to Game_ID in the BB_MULTIGAME_DESC table. If not a NULL value or zero, then it indicates that this Slot Type is a multi-game machine. 
ElecCoinIn    numeric(15, 2)  NO  Amount in Dollars of the Electronic Coin In of the Game based on electronic meters during the corresponding period (e.g. CUR, PTD, MTD, QTD, YTD, LTD). 
ElecCoinOut    numeric(15, 2)  NO  Amount in Dollars of the Electronic Coin Out of the Game based on electronic meters during the corresponding period (e.g. CUR, PTD, MTD, QTD, YTD, LTD). 
ElecJackpot    numeric(15, 2)  NO  Amount in Dollars of the Electronic Jackpots of the Game based on electronic meters during the corresponding period (e.g. CUR, PTD, MTD, QTD, YTD, LTD). 
GameStarts    int  NO  Meter that shows the lifetime number of times the game was played on a machine. 
GamesWon    int  NO  Meter that shows the lifetime number of times the game was won playing this game on a machine. 
DaysOnLine    int  NO  The number of days the game was on-floor with an active status.
