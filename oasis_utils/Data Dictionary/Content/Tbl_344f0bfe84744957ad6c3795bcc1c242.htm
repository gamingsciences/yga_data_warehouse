dbo.CDS_SlotMast 
Logical Entity Name  CDS_SLOTMAST 
Default Table Name  CDS_SlotMast 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Main table to keep track of the configurations of the slot machines. For each unique slot mast ID, the machine can have several configurations during its life. The revision number from the machine identifies each configuration.

SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.

Sen    smallint  NO  Unique ID number used to identify the Sentinel located within a slot machine. 
OasisId    char(9)  YES  The OASIS ID is the electronic address of the game. It is a combination of the 2-digit site number if the gaming organization has multiple physical sites, the 3-digit DPU number, and the 2-digit Sentinel number (Site-DPU-Sentinel e.g., 01-101-31). An OASIS ID of 00-00-00 indicates the machine is off line. The OASIS ID is required for every game and you can assign it through the Machine Notebook. You must change this ID in BlackBart when you move a machine to a different site, DPU, or Sentinel.

SlotTypeAlpha    char(10)  YES  A unique Slot Type identifier based on Alphabetic characters.

Loc    char(5)  YES  Casinos divide the gaming floor in to different areas, each area is then divided up into small unites called sections, each section in turn is further divided up into small units called locations. Each location is then given a unique identifier.
