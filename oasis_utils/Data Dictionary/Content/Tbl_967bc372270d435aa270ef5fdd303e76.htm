Default Table Name  WIS_NETWIN_PTD_EXTRACT 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Persistent temporary table used for interim processing of data PTD (Period (Week) To Date) period data for Wisconsin Net Win extracts. All data from WIS_NETWIN_xxx_EXTRACT tables are populated and massaged and inserted/updated in WIS_NETWIN_EXTRACT_TMP, which is the final source of the data extract.
