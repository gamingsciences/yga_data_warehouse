dbo.PlayerAddress 
Logical Entity Name  PlayerAddress

Physical Only  NO

Owner  dbo 
Definition  Player Addresses (by Player ID). The addresses containing a "1" in the column name are mailing address components and the addresses with a "2" in the column name are home, or residence address components.
