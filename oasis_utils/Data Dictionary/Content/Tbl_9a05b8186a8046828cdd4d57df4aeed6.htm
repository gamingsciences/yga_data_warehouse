dbo.SMAPromotionLicense Columns 
ColumnName  Domain  Datatype  NULL  Definition 
LicenseID    int  NO  PK - Int value that is generated by ATI. Each number represents one unique license. 
EventTypeID    tinyint  YES  TinyInt value that is generated by ATI and defined in the SpeedMediaEventType table. Values can include, among others, Splashdown Countdown, Ricochet Rewards, Jackpot Announcement, CDM Promotion and Totalizer Announcement. 
VendorID    smallint  NO  SmallInt value that is generated by ATI and defined in the SMAVendor table. Each number identifies a unique vendor. 
Name    nvarchar(64)  NO  nVarChar(128) text field which lists the name of the license.
