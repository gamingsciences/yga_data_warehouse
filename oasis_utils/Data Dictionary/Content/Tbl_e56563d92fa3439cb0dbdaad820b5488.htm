dbo.SpeedMediaTriggerAward Columns 
ColumnName  Domain  Datatype  NULL  Definition 
TriggerAwardID    int  NO  PK - Int value that is foreign keyed to an identity field in the SpeedMediaTriggerAward table. Each number uniquely identifies a trigger or award transaction. 
EventID    int  NO  Int value that is foreign keyed to an identity field in the SpeedMediaEvent table. Each number uniquely identifies an event. 
TriggerTypeID    tinyint  YES  TinyInt value generated by ATI and defined in the SpeedMediaTriggerType table. It describes each Trigger as being of a specific type (e.g. Coin In, Coin Loss, All Jackpots, Attendant Paid Jackpot, etc.). 
TriggeredDateTime    datetime  NO  DateTime value indicating when a trigger was initiated on the Sentinel. 
AuditDate    datetime  NO  DateTime value which is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
PlayerID    int  YES  Int value that is defined in the CDS_Player table. Each number identifies a player. 
SlotMastID    int  YES  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.  
Revision    smallint  YES  SmallInt value which is a system ID assigned by BlackBart when a slot machine record is added to or modified in the system. Each time a slot machine record is opened, modified in any way, and then saved through BlackBart, the revision number is incremented by +1. 
AwardTypeID    tinyint  NO  TinyInt value that is defined in the SpeedMediaAwardType table. Each number associates one Ricochet Event to a specific award type. 
AwardedAmount    numeric(15, 2)  YES  Numeric(15,2) value indicating the amount awarded or won. The value can be in points, dollars, or whatever type is defined for this row in the AwardTypeID column. 
PtsAdjustmentID    int  YES  Int value which references the Adjustment_ID in CDS_Adjustment when points are won as a promotion award. 
EFTTransID    int  YES  Int value which references the EFTTransactionID in EFTTransaction when PBT Promo or PBT Cash are won as a promotion award. 
TriggeredByID    int  YES  Int value that, when null, is an indication that the current row is a trigger. When not null, it indicates the current row is an award and the value in this field points back to the TriggerAwardID of the row that triggered it. 
AwardStatusID    tinyint  NO  Award Status ID as defined in SMAAwardStatus Table 
ExternalID    bigint  YES  Identifier used when awarding to the EGM so that when a confirmation comes back from the game that something was awarded, either the SpeedMedia application or a thrid party that awarded to the game can match it up with a local record. 
RowVersion    timestamp  NO  Versioning information used for concurrency checking.
