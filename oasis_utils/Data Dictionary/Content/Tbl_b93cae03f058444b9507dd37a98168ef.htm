dbo.SplashdownCountdownEvent Columns 
ColumnName  Domain  Datatype  NULL  Definition 
EventID    int  NO  PK - Int value that is foreign keyed to an identity field in the SpeedMediaEvent table. Each number uniquely identifies an event for Splashdown Countdown. 
CountdownTriggerTypeID    tinyint  NO  TinyInt value that is defined in the SpeedMediaTriggerType table. Each number associates one Splashdown Countdown Event to a specific trigger type. 
CountdownTriggerThreshold    money  NO  Money value indicating the whole dollar amount which triggers the start of a Splashdown Countdown event. 
CountdownTimer    int  NO  BA requires that the machine list will not be updated during a Countdown Timer. 
CountdownTriggerAwardTypeID    tinyint  NO  TinyInt value that is defined in the SpeedMediaAwardType table. Each number associates one Splashdown Countdown Event to a specific award type. 
CountdownTriggerAwardAmount    int  NO  Int value indicating the amount awarded or won. The value can be in points, dollars, or whatever type is defined for this row in the AwardTypeID column. 
CountdownWinnerTriggerTypeID    tinyint  NO  TinyInt value generated by ATI and defined in the SpeedMediaTriggerType table. It describes each Trigger as being of a specific type (e.g. Coin In, Coin Loss, All Jackpots, Attendant Paid Jackpot, etc.). 
CountdownWinnerTriggerThreshold    money  NO  Money value which triggers the winner during a Splashdown Countdown. 
CountdownWinnerTriggerAwardTypeID    tinyint  NO  TinyInt value that is defined in the SpeedMediaAwardType table. Each number associates one Splashdown Countdown Event to a specific award type. This specifies the type of award granted when a winner is triggered during a Splashdown Countdown. 
CountdownWinnerTriggerAwardAmount    int  NO  Int value indicating the amount awarded or won. This specifies the award amount granted when a winner is triggered during a splashdown countdown. 
Personal    bit  NO  Bit value indicating if the Splashdown Countdown is a personal promotion. When a personal Splashdown is triggered, the person who triggered the promotion is in the countdown by themselves. 
EventStartAccumulationTypeID    tinyint  NO  TinyInt value which defines how trigger threshold values will be accumulated. 
AwardStartAccumulationTypeID    tinyint  NO  TinyInt value which defines how award threshold values will be accumulated.
