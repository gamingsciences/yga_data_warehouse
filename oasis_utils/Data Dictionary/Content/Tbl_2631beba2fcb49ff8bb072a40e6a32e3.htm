dbo.SMANotificationMeter 
Logical Entity Name  SMANotificationMeter 
Default Table Name  SMANotificationMeter 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  This table defines the state of the meters and thresholds of the CDM Event at the time of the award. It is the equivalent to a TriggerAward table for CDM.
