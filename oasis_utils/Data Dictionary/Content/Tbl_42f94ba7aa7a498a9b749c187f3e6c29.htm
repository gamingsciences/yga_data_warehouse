AuditDate    smalldatetime  NO  The Audit Date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24-hours following the Slot Gaming Day Start set in OASIS Administrator. 
AuditStatusID    tinyint  NO  Lookup ID for the Audit Status into the AuditHistoryStatuses lookup table. 
NewPTD    bit  NO  Flag set to 'Y' to indicate that a new Period-to-Date (PTD) period is starting at this particular audit date. PTD is started new every # of consecutive calendar days or calendar months. 
NewMTD    bit  NO  Flag set to 'Y' to indicate that a new Month-To-Date (MTD) period is starting at this particular audit date. MTD is started new every # of consecutive calendar days or on the 1st day of a new calendar month. 
NewQTD    bit  NO  Flag set to 'Y' to indicate that a new Quarter-To-Date (QTD) period is starting at this particular audit date. QTD is started new every # of consecutive calendar days or months. 
NewYTD    bit  NO  Flag set to 'Y' to indicate that a new Year-To-Date (YTD) period is starting at this particular audit date. YTD is started new every # of consecutive calendar days or months.
