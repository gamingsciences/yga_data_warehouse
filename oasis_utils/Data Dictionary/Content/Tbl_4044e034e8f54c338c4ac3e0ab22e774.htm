dbo.DM_BUFFER 
Logical Entity Name  DM_BUFFER 
Default Table Name  DM_BUFFER 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  In Diagnostic Monitor, when the User clicks on the "Transaction" tab, Diagnostic Monitor does a Select from CDS_BUFFER and inserts the data into the DM_BUFFER table. The DM program then selects the data from the DM_Buffer table so it can be displayed and sorted on the screen in Diagnostic Monitor.
