Definition  The OASIS ID is the electronic address of the game. It is a combination of the 2-digit site number if the gaming organization has multiple physical sites, the 3-digit DPU number, and the 2-digit Sentinel number (Site-DPU-Sentinel e.g., 01-101-31). An OASIS ID of 00-00-00 indicates the machine is off line. The OASIS ID is required for every game and you can assign it through the Machine Notebook.
