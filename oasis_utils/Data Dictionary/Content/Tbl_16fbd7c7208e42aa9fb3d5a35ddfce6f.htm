dbo.CDS_JFPend 
Logical Entity Name  CDS_JFPEND 
Default Table Name  CDS_JFPend 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  The Poller inserts pending Jackpot/Fill transactions into this temporary storage table. Once a transaction is processed, Jackpot/Fill removes it; or if a transaction is not processed in a certain period of time, the Poller moves it.

If this setting is checked in OASIS Administrator for this particular Machine, the stored procedure RT_JFPEND1 will combine the current Jackpot meter and the amount of the current jackpot to get a "Calculated Jackpot Meter". This is required only in Colorado. 
CalcCCMeter    bigint  NO  Used in conjunction with Maximum Jackpot Amount. If the cashout is greater than maximum amount allowed for this slot machine, it becomes a cancelled credit. This is a calculated value and depends on the setting in OASIS Administrator-Setup-Poller-Sentinel Options called "Add Cur CC". If this setting is checked in OA for this particular Machine, the stored procedure RT_JFPEND1 will combine the current cancelled credit meter and the amount of the current jackpot to get a "Calculated Canceled Credit Meter". This is required only in Colorado. 
TotalHandPay    bigint  NO  This is the "Total Hand Pay" meter that is sent to the system from the slot machines. The meter is not a true "Total Handpay" meter so it is seldom used at the moment. In the future, slot machines will send a true "Total Hand Pay" meter including all hand pay lockups.
