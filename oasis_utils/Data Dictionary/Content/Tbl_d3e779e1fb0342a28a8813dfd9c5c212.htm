dbo.BB_BadScale Columns 
ColumnName  Domain  Datatype  NULL  Definition 
AuditDate    smalldatetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
LoadTime    datetime  NO  The date and time when this transaction was inserted into this table. 
ScaleID    int  YES  Indicates the type of scale that was used. To make this determination match this ScaleID with the ScaleType_ID in the BB_SCALETYPE table which will show you the description of the scale. 
ScaleFileName    varchar(255)  YES  The name of the input table used to load the coins. 
ScalePathName    varchar(255)  YES  Unused field. 
SlotNumber    int  NO  Slot Number refers to the number of the gaming machine. This number corresponds to the number on the machine's cabinet. 
RecordNumber    int  NO  Record number of this transaction as its stored in the input table. 
Amount    numeric(12, 2)  NO  Dollar amount of this transaction. 
CounterNumber    tinyint  YES  A number that identifies the counter number from which an coin scale was loaded.
