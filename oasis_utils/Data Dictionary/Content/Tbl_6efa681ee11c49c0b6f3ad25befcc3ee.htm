dbo.GEN_Country

Physical Only  NO

Owner  dbo 
Definition  Lookup table of country descriptions relating to a 3-letter code. Typically used to display a long country name for the User when selecting a Player's associated country while storing a smaller, unique 3-letter code in the database.
