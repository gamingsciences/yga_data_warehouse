dbo.SpeedMediaNonTriggeredEvent 
Logical Entity Name  SpeedMediaNonTriggeredEvent 
Default Table Name  SpeedMediaNonTriggeredEvent 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  This table contains a record of events that were scheduled and active, but never triggered. 
Note  This table stores all the eventID occured at the time when a Triggered ID from SpeedMediaTriggered occured. this ratio between the EventIDs and Triggered ID gives Casino Operator a way to measure the how many loss on each winning event. 
PreSQL   
DDL Code  CREATE TABLE dbo.SpeedMediaNonTriggeredEvent(
     ID                      int     IDENTITY(1,1),
     TriggeredID             int     NOT NULL,
     NonTriggeredEventID     int     NOT NULL,
     CONSTRAINT pknc_SpeedMediaNonTriggeredEvent_ID PRIMARY KEY NONCLUSTERED (ID)
     WITH FILLFACTOR = 80,
     CONSTRAINT fk_SpeedMediaNonTriggeredEvent_SpeedMediaEvent_EventID FOREIGN KEY (NonTriggeredEventID)
     REFERENCES dbo.SpeedMediaEvent(EventID),
     CONSTRAINT fk_SpeedMediaNonTriggeredEvent_SpeedMediaTriggerAward_TriggerAwardID FOREIGN KEY (TriggeredID)
     REFERENCES dbo.SpeedMediaTriggerAward(TriggerAwardID)
)
go
