Event_ID    int  NO  The event_id for the Event. 
EventName    varchar(50)  NO  The name of the Event. 
EventType    int  YES  Used to indentify the type of event. Numbered or unnumbered. 
RangeType    int  YES  There are different ways of selecting players to an Event. One is by using a date range. There are different date ranges to select from: Date Last Played, Birthdate, Anniversity, Enrollment Date, and Date Earned with the amount of points earned. This value is used to indicate the which date range is used by the event: 0 - Date Last Played, 1 - Birthdate, 2 - Anniversity, 3 - Enrollment Date, and 4 - Date Earned 
RangeStartDate    datetime  YES  There are different ways of selecting players to an Event. One is by using a date range. There are different date ranges to select from: Date Last Played, Birthdate, Anniversity, Enrollment Date, and Date Earned with the amount of points earned. This is begin date of the date range. 
RangeEndDate    datetime  YES  There are different ways of selecting players to an Event. One is by using a date range. There are different date ranges to select from: Date Last Played, Birthdate, Anniversity, Enrollment Date, and Date Earned with the amount of points earned. This is end date of the date range. 
Points    int  YES  This value is used to select players to be assoicated with an event 
EventStatus_ID    smallint  YES  The current status of the event. 
LocationRestriction    smallint  YES   
PlayerSelectionMethod    smallint  YES  A numeric indicator if the event has associated or non-associated players. 0 - is associated players, 2 - is non-associated players. 
NonUniqueQty    int  YES  A numeric value indicating the maximum number of entry redemptions and regular redemptions allow for an event. This value is only use for Event that would use NonUnique redemption numbers. This value will be zero if the Event uses unique redemption numbers.
