Unmerge.StatPointCalc 
Logical Entity Name  STATPOINTCALC 
Default Table Name  StatPointCalc 
Physical Only  NO 
Do Not Generate  False 
Owner  Unmerge 
Definition  Ratings that are summed together based on the PointCalc_ID for a player that was merged with another player. Updating is based on the values in PlayerID, StatType, and PointCalc_ID. The column PointCalc_ID is derived from the GEN_TABLEGAME table, which is based on the Player's skill level ('Soft', 'Medium', or 'Hard') and game speed ('Slow', 'Medium', or 'Fast'). These records were recorded prior to the merge to facilitate an unmerge.

The column was kept for backwards compatibility and may be removed in a future release.
