dbo.BB_Location Columns 
ColumnName  Domain  Datatype  NULL  Definition 
LocID    smallint  NO  Unique ID number for location. 
Description    varchar(30)  NO  Description or name of location. 
Site    smallint  YES  Casino floor site number. 
DPU    smallint  YES  Unique ID number for the DPU associated with up to 31 Sentinels. 
Sen    smallint  YES  Unique ID number used to identify the Sentinel located within a slot machine. 
CasinoCode    char(2)  YES  A component of the Location assigned to a machine. Casino Code can represent a single casino or distinguish between casinos in a multiple casino environment. 
Area    char(5)  YES  Casinos divide the gaming floor in to different areas and give each area an unique identifier. 
Section    char(5)  YES  Casinos divide the gaming floor in to different areas. Each area is then divided into smaller units called sections. Each section is then given a unique identifier. 
Loc    char(5)  YES  Casinos divide the gaming floor in to different areas. Each area is then divided up into small unites called sections. Each section in turn is further divided up into small units called locations. Each location is then given a unique identifier. 
Status_ID    smallint  YES  ID number associated with status from BB_Status table. 
Bank_ID    smallint  YES  ID number associated with slot bank from BB_SlotBank table. 
CheckDuplicity    char(1)  NO  Check for duplicate location. 'Y' = Yes, 'N' = No.
