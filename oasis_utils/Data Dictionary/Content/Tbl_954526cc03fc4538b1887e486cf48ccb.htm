Definition  Defines the types of awards available to the system to use when awarding. Formerly named SpeedMediaAwardType.  
Note  Types are:
Points
Point Multiplier
PBT Promo Dollars
PBT Cash
No Award.

The Point Multiplier is effective for points earned within the start and the end of the countdown timer.
