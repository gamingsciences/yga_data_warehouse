dbo.BB_BadJackpotFill Columns 
ColumnName  Domain  Datatype  NULL  Definition 
JackpotFill_ID    int  NO  Unique ID assigned to each transaction as it is inserted into this table. This is an Identity field and the Primary Key. 
Booth_ID    smallint  NO  A unique numeric value from 0 - 99 to identify a particular Jackpot/Fill workstation. 
AuditDate    smalldatetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
LoadTime    smalldatetime  NO  The date and time when this transaction was inserted into this table. 
TransType    char(1)  NO  Identifies the transaction type which may be either 'J' for jackpot or 'F' for fill. **Note:** 'F' is also used to indicate transaction types for PMFills. 
SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.  
Revision    smallint  NO  Each time a slot machine record is opened, modified in any way, and then saved through BlackBart, the revision number is incremented by +1. 
SlotNumber    int  NO  Slot Number refers to the number of the gaming machine. This number corresponds to the number on the machine's cabinet. 
Denomination    numeric(12, 2)  NO  The monetary value a slot machine is setup to use. 
Amount    numeric(12, 2)  NO  Dollar amount of the jackpot or fill transaction. This column is zero if this transaction is a promotional jackpot. 
JPPromo    numeric(12, 2)  YES  Contains the dollar amount. If this is a promotional jackpot transaction, otherwise this column is zero.
