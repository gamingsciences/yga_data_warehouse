dbo.Tiered_RWD_Profiles Columns 
ColumnName  Domain  Datatype  NULL  Definition 
Profile_ID    int  NO  Unique numeric identifier for this profile. 
Name    nvarchar(50)  YES  User defined name given to the profile. 
PlayPeriodTimeRange    tinyint  YES  Flag indicating which play period was chosen (session, lifetime etc.) 
PlayPeriodTripChoice    tinyint  YES  If the period of play is Trip, then this indicates which trip choice to use (see TIERED_RWD_LKP_TRIPCHOICE) 
PlayPeriodTimeStart    datetime  YES  Date / time when the play period begins (the time period for which play is calculated on for reward eligibility). 
PlayPeriodTimeEnd    datetime  YES  Date / time when the play period ends (the time period for which play is calculated on for reward eligibility). 
IssueOption    tinyint  YES  Indicates whether to issue all rewards earned, or just the highest one obtained. 
RewardStatChoice    tinyint  YES  Indicates which stat choice to use for calculating reward elgibility (such as Total In). 
ExpireHours    smallint  YES  How many hours until issued rewards are automatically expired. 
LastModified    datetime  YES  Date/time of the last edit/change made to this profile. 
Deleted    bit  YES  Flag indicating whether or not this profile has been "deleted". Deleted in this sense means no longer shown to the user - it's never deleted from the database. 
IncludeFlag    smallint  YES  Flag to indicate whether groups are included, ranks are included, non group players and non ranked players. 
ProfileStartTime    datetime  YES  Date/time at which point the profile starts issuing rewards. 
ProfileEndTime    datetime  YES  Date / time at which point the profile no longer issues rewards. 
AllowMultipleSamePrizeIssue    bit  YES  The Allow multiple issuance of the same prize allows the same prize to be redeemed more than once, up to the maximum number. 
MaxNumberPrizes    int  YES  The optional Maximum number of rewards field limits the number of rewards that can be redeemed. 
RowVersion    timestamp  NO
