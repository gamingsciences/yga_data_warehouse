dbo.EFT_Session 
Logical Entity Name  EFT_SESSION 
Default Table Name  EFT_Session 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  This table holds transactions that occur while a player card is inserted and then removed. It is not related to EFT or PersonalBanker. If a player has their card inserted and then does any one or more of 13 different transactions (such as insert a bill or ticket), a "Cash Transfer tracking" transaction is sent to the system and the data is inserted into this table for reporting purposes.
