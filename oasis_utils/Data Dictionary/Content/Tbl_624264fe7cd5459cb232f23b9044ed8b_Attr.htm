DateUpdated

Attribute Name  DateUpdated 
ColumnName  DateUpdated 
Logical Rolename   
RoleName   
Domain   
Physical Only  NO 
Definition  This is the date that this Campaign was Updated. Note: If any Offers or Coupons UNDER THIS CAMPAIGN are added or Changed, then the CAMPAIGN is considered to be updated. The Program uses DateUpdated and iUserWhoUpdated to avoid conflicts. Before a change is made to a Campaign (or underlying Offer or Campaign) the program makes sure that a no one else has made a change to the campaign since the last time data was read. If there was a change, it stops. If there wasn't a change, it makes the change, then sets these 2 values.
