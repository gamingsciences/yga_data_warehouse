Definition  [Marketing] Campaigns are, at the lowest level, media files running on the sentinel. They are a grouping of more and more granular items that eventually reaches the individual file level. A campaign contains one or more packages. A package contains one or more screens. A screen contains one or more media files. Some types of media files include Image, Video, Stream and Flash. 
Note 
This table describes a SpeedMedia Campaign.
