dbo.JF_Workstation 
Logical Entity Name  JF_WORKSTATION 
Default Table Name  JF_Workstation 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  This table holds one record for each Jackpot/Fill station that has been setup. Defines data referencing default report printers, booth & window setup, casino ID, and number of card readers.
