dbo.BB_BadBill Columns 
ColumnName  Domain  Datatype  NULL  Definition 
BillTx_ID    int  NO  Unique ID assigned to each transaction as it is inserted into this table. This is an Identity field and the Primary Key. 
AuditDate    smalldatetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
SDrop_ID    int  NO  Transaction number as it appears in the input table. 
EnteredDateTime    smalldatetime  NO  The date and time when the transaction was inserted into this table. 
SlotNumber    int  YES  Slot Number refers to the number of the gaming machine. This number corresponds to the number on the machine's cabinet. 
Cash1    smallint  NO  Meter that shows the lifetime number of one dollar bills inserted into the machine. 
Cash2    smallint  NO  Meter that shows the lifetime number of two dollar bills inserted into the machine. 
Cash5    smallint  NO  Meter that shows the lifetime number of Five dollar bills inserted into the machine. 
Cash10    smallint  NO  Meter that shows the lifetime number of ten dollar bills inserted into the machine. 
Cash20    smallint  NO  Meter that shows the lifetime number of twenty dollar bills inserted into the machine. 
Cash50    smallint  NO  Meter that shows the lifetime number of fifty dollar bills inserted into the machine. 
Cash100    smallint  NO  Meter that shows the lifetime number of hundred dollar bills inserted into the machine. 
CashMutilated    smallint  NO  Number of bills mutilated and reported by the Sentinel.
