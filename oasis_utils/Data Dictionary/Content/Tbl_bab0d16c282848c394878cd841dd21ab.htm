dbo.SMAAwardEGMBonusPayout Columns 
ColumnName  Domain  Datatype  NULL  Definition 
TriggerAwardID    int  NO  The SMA Trigger ID 
EGMBonusPayoutTypeID    tinyint  NO  Record ID 
TransactionID    varchar(20)  NO  Unique number generated when the record is inserted into the table. This is the Transaction ID sent with the SAS poll to do AFT bonusing. 
CoinIn    bigint  YES  The Oasis Coin In meter value at the time the Crazy Hits Award was awarded. This value will be the same before & after the award. 
CoinOut    bigint  YES  The Oasis Coin Out meter value at the time the Crazy Hits Award was awarded. This value will be the same before & after the award. 
ProgessiveCoinIn    bigint  YES  The Oasis ProgCoinIn meter value (meter 61) at the time the Crazy Hits Award was awarded. This value will be the same before & after the award. 
InitialExternalBonusingAttendantPayout    bigint  YES  The Oasis AttPaidExtBonus meter value (meter 78) prior to the Crazy Hits Award being awarded. 
InitialExternalBonusingMachinePayout    bigint  YES  The Oasis MachPaidExtBonus meter value (meter 79) prior to the Crazy Hits Award being awarded. 
CurrentExternalBonusingAttendantPayout    bigint  YES  The Oasis AttPaidExtBonus meter value (meter 78) after the Crazy Hits Award is awarded. 
CurrentExternalBonusingMachinePayout    bigint  YES  The Oasis MachPaidExtBonus meter value (meter 79) after the Crazy Hits Award is awarded. 
PaidPlayerID    int  YES  The player ID that a Crazy Hits award was actually paid to at the Sentinel. This ID can be different than the PlayerID column in the SpeedMediaTriggerAward table if the player who placed the winning wager removed their card before the win was paid to the EGM. 
PaidAmount    numeric(15, 2)  YES  The actual amount paid to the player (PaidPlayerID) for the Crazy Hits award. This amount can be different than the AwardedAmount column in the SpeedMediaTriggerAward table due to the requirement of making the actual awarded amount evenly divisible by the meter denomination of the EGM. 
RowVersion    timestamp  NO  Versioning information used for concurrency checking.
