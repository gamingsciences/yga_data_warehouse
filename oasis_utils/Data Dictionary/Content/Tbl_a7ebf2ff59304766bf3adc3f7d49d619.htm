dbo.PB_Revenue 
Logical Entity Name  PB_REVENUE 
Default Table Name  PB_Revenue 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Stores the most current revenue values and totals after a soft count has been done for a gaming day. It sums all the values stored in PB_REVENUEDTD. Every insert first deletes all records for the given date and the recalculates all relevant totals again.
