dbo.SpeedMediaEventCategory

Owner  dbo 
Definition  This table links the SpeedMediaCategory table to the SpeedMediaEventType table and identifies which event types belong with which category. For example, the event types Ricochet Reward and Splashdown Countdown belong to the Promotion category.
