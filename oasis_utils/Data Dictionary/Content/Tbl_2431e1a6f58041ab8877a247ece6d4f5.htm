dbo.SchedulerAuditTrail Columns 
ColumnName  Domain  Datatype  NULL  Definition 
AuditTrailID    int  NO  Primary Key 
SchedulerInfoID    smallint  NO  References SchedulerInfoID from the SchedulerInfo table. Specifies which scheduled job this audit trail entry refers to 
ServiceID    smallint  NO  References ServiceID from table SchedulerServices. Identifies the kind of service being run for which this audit trail entry has been created 
FrequencyID    tinyint  NO  References FrequencyID from table ScheduleFrequency. 1=Daily, 2=Weekly, 3=Monthly 
IsActive    bit  NO  Identifies whether the service is active 
LastRunDateTime    datetime  NO  DateTime on which the service was last run 
IntervalMonths    tinyint  YES  Will be non-zero only if FrequencyID is 3, i.e. monthly. Specifies the interval in months at which the service will run recurringly 
IntervalDays    tinyint  YES  Specifies day of month on which job will run when the job is set to run monthly but does not have scheduled days of the week 
ScheduleTime    time(7)  YES  Time at which the job should run. Will be populated in case the job is non-recurring 
IntervalInMins    int  YES  Specifies the interval in minutes at which the job should run, on the days it is scheduled to run 
StartDate    date  NO  Date on which the schedule will start 
EndDate    date  YES  Date on which the schedule will end 
StartTime    time(7)  NO  Time at which the schedule will start, on the days it is scheduled to run 
EndTime    time(7)  YES  Time at which the schedule will end, on the days it is scheduled to run 
ScheduledDescription    varchar(255)  NO  Description of the schedule, describing its run frequency 
UpdateDateTime    datetime  NO  DateTime when the schedule was updated 
UserID    int  NO  Id of User that made changes to the schedule
