dbo.Resync 
Logical Entity Name  Resync 
Default Table Name  Resync 
Physical Only  NO 
Do Not Generate  False 
Owner  dbo 
Definition  Contains records that were popped off the Service Broker queue but could not be processed causing a record (e.g. patron update or config record update) to be out of sync between central and one or more local properties.
