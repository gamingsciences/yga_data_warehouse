dbo.SMAScreenType

Physical Only  NO

Owner  dbo 
Definition  Formerly named ScreenType, this table contains each valid Type that can be associated to a Screen (e.g. Welcome, Congratulations, Birthday, Anniversary, etc.). 
Note  The coordinate used to be TOP, LEFT, RIGHT, and BOTTOM. However, these are SQL keyboards, so they are changed to:

Y1 is top
X1 is left
X2 is right
y2 is bottom
