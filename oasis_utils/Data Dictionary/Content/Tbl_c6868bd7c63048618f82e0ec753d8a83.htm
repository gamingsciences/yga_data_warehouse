dbo.BB_RevenueInfo Columns 
ColumnName  Domain  Datatype  NULL  Definition 
SlotMast_ID    int  NO  A auto-generated system ID assigned by BlackBart when a slot machine is added to the CDS_Slotmast table.  
AuditDate    smalldatetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
CashPulledDT    smalldatetime  YES  Cash pulled date. 
CoinDroppedDT    smalldatetime  YES  Coin dropped date. 
MasterID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table, that represents the cumulative of all the meter snapshots that occurred for this slot and auditdate to represent what was used for calculating revenue. 
RollID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used for rollover revenue calculation. 
GameDayEndID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used for game-day end revenue calculation. 
HardDropID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used for hard drop revenue calculation. 
SoftDropID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used for soft drop revenue calculation. 
PrevMasterID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table, that represents the cumulative of all the meter snapshots that were used as the prior snapshot for calulating revenue. 
PrevRollId    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used as the prior for rollover revenue calculation. 
PrevGameDayEndID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used for the previous meters for game-day end revenue calculation. 
PrevHardDropID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used for the prior hard drop meter during revenue calculation. 
PrevSoftDropID    bigint  YES  Uniquely identifies a Meter snapshot in the RT_MeterDrop table that was used for prior meters for soft drop revenue calculation. 
CoinDropPosted    bit  NO  Flag indicating if CoinDrop was posted to this machine. 
BillDropPosted    bit  NO  Flag indicating if Bills where posted to this machine. 
VoucherDropPosted    bit  NO  Flag indicating if Vouchers where posted to this machine. 
CashCouponDropPosted    bit  NO  Flag indicating if Cash Coupons where posted to this machine. 
PromoCouponDropPosted    bit  NO  Flag indicating if PromoCoupons where posted to this machine.
