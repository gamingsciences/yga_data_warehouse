dbo.BB_Ticket_Load Columns 
ColumnName  Domain  Datatype  NULL  Definition 
TicketRecNum    int  NO  This record number is the same as it would be for the ticket in cds_ticket. 
ValidationNumber    varchar(32)  NO  A randomly-generated 8- or 18-digit number (depending on the slot machine). 
TicketNumber    int  NO  A slot machine-generated sequential number used to identify the number tickets printed by that slot machine. 
PrintedSlotMastID    int  NO  This is the Slotmast ID of the machine that printed the ticket. 
PrintedMachineNumber    int  NO  This is the number of the slot machine that printed the ticket. 
TicketStatus    tinyint  NO  The status of this ticket transaction. 1 = Printed, 2 = Printed, 4 = Redeemed, 5 = Expired, 6 = Voided. 
Amount    money  NO  Dollar amount of this transaction. 
AuditDate    datetime  NO  Audit date is an accounting reference to a "gaming day" that is in or through the audit process. Gaming day is defined by a period of 24 hours following the Slot Gaming Day Start set in OASIS Administrator. 
LoadedDateTime    datetime  YES  The date and time that this ticket transaction was loaded into BlackBart.
