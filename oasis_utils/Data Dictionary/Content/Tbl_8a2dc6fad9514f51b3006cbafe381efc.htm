dbo.SpeedMediaMachineGroupCriteria Columns 
ColumnName  Domain  Datatype  NULL  Definition 
CriteriaID    int  NO  PK - Int value that is automatically generated (identity). Each unique number represents one machine group criteria. 
MachineGroupID    int  NO  Int value that is foreign keyed to an identity field in the SpeedMediaMachineGroup table. Each number uniquely identifies a SpeedMedia Machine Group. 
SlotNumber1    int  YES  Int value which indicates the beginning of the range of Slot Numbers to include in the criteria list. 
SlotNumber2    int  YES  Int value which indicates the ending of the range of Slot Numbers to include in the criteria list. 
SlotType_ID1    int  YES  Int value which indicates the beginning of the range of Slot Type to include in the criteria list. 
SlotType_ID2    int  YES  Int value which indicates the ending of the range of Slot Type to include in the criteria list. 
CasinoCode    char(2)  YES  Char(2) value which is a component of the Location assigned to a machine. Casino Code can represent a single casino or distinguish between casinos in a multiple casino environment. 
Section    char(5)  YES  Char(5) value which is derived when casinos choose to divide the gaming floor in to different areas. Each area is then divided up into small units called sections. Each section is then given a unique identifying value. 
Denomination    numeric(12, 2)  YES  Numeric(12,2) value indicating the monetary value which equates to one credit of play. 
Area    char(5)  YES  Char(5) value which is derived when casinos segment the gaming floor. Each area is given a unique identifying value. 
Loc    char(5)  YES  Char(5) value derived when casinos divide the gaming floor in to different areas. Each area is then divided up into smaller units called sections and each section in turn is further divided up into smaller units called locations. Each location is then given a uniquely identifying value. 
MachineNotebookGroupID    char(5)  YES  Char(5) value which represents the Black Bart Machine Notebook GroupID. 
BlackBartSlotGroupID    int  YES  Int value which uniquely identifies a Black Bart slot group in the BB_SlotGroup table. 
StyleID    smallint  YES  SmallInt value which is defined in the BB_Style table. Values can include, among others, Reel Slot, Video Slot and Video Poker. 
ManufacturerID    smallint  YES  SmallInt value which is defined in the BB_MFR table. Values correspond to a slot machine manufacturer.
